# JS/TS .conf File Parser

### installation
```console
npm i js-conf-parser
```

### usage
```js
// with absolute path
parseConfig("/path/to/config", "filename.cfg")

// with relative path
parseConfig("path/to/config", "filename.cfg")
```


for the supported features, please check the [examples](https://github.com/eliteSchwein/js-conf-parser/tree/main/testing)
